# Game of life

Learned Rust by reimplementing [Conway's Game of life](https://en.wikipedia.org/wiki/Conway%27s_Game_of_Life).

## Getting started

```bash
git clone git@github.com:lvndry/game-of-life.git
cd game-of-life
./start.sh
```
